1.2.1. XHTML：新热点

HTML 有块级元素(如div、p、table等)和内联元素(如a、em、strong等)之分，内联元素永远不能包含块级元素。

XHTML 特点

  • DOCTYPE 声明
  • 保持标记具有良好的架构
  • 关闭每一个元素
  • 把元素和属性值设置为小写
  • 必须为每个属性指定一个值
